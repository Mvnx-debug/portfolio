Clonagem do YouTube
Visão Geral

Este projeto consiste em uma página web que simula uma seção de vídeos similar à aba de vídeos do YouTube, desenvolvida como parte do curso de CSS da plataforma DIO (Digital Innovation One). A implementação foi feita utilizando apenas HTML e CSS puros, sem o uso de frameworks ou bibliotecas JavaScript.

Objetivos
  • Criar uma interface limpa e responsiva para exibição de vídeos
  • Praticar conceitos avançados de CSS como Flexbox e Grid Layout
  • Implementar um design semelhante ao YouTube mantendo identidade própria
  • Desenvolver habilidades de posicionamento e estilização CSS

Automação de Extração de Dados de PDF
Visão Geral

Este projeto consiste em um script em Python que automatiza a leitura e extração de dados de arquivos PDF, especificamente voltado para documentos de pedidos em uma empresa. O objetivo principal foi agilizar o processamento desses documentos, eliminando a necessidade de digitação manual e reduzindo erros humanos.

Funcionalidades Implementadas:
  • Leitura de PDF → Utilização de bibliotecas como PyPDF2 ou pdfplumber para extrair texto de arquivos PDF.
  • Processamento de Dados → Filtragem e organização das informações relevantes (como números de pedido, clientes, valores e datas).
  • Saída Estruturada → Geração de um arquivo Excel (XLSX) com os dados extraídos, pronto para análise.
  • Automatização Mensal → Configuração para rodar periodicamente, auxiliando no fechamento de relatórios.
  • Disponivel no meu github
Tecnologias Utilizadas:
  • Python
  • PyPDF2 / pdfplumber
  • Pandas
  • OpenPyXL
